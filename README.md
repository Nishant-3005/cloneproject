# 🏡 Wanderlust

A full-stack Airbnb-inspired web application where users can explore, create, edit, review, and manage travel listings.

## 🚀 Live Demo

https://cloneproject-g775.onrender.com
## 📌 Features

- 🔐 User Authentication (Sign Up/Login)
- 🔒 Authorization using Passport.js
- 🏠 Create, Edit & Delete Listings
- ☁️ Cloudinary Image Upload
- ⭐ Review & Rating System
- 🔍 Search Listings
- 💬 Flash Messages
- 📱 Responsive Design
- 🗂 MVC Architecture

## 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Passport.js
- EJS
- Bootstrap 5
- Cloudinary
- Multer
- Joi
- Express Session
- Connect Flash
